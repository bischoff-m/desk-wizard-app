# Desk Wizard

Low resource dpi-aware interactive wallpaper desktop app. Built in window system and
display manager.

Built using tauri and nextjs. Needs WallpaperEngine or Lively Wallpaper installed.

## Setup

- Add .env

```bash
npm install
npm run tauri dev
```
